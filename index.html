Engineering World Health - Alzheimer's Detection Device

As part of the programming team, I helped develop a functional software system for a low-cost, non-invasive device that aims to detect Alzheimer’s disease in its early stages. Using hyperspectral imaging of the retina, the device identifies amyloid aggregates, an early biomarker of Alzheimer’s, allowing for earlier diagnosis and more accurate tracking of disease progression.

Mondo Bot

Mondo Bot is a Discord bot inspired by Discord's Trick-or-Treat bot. It runs timed challenge events where users race to respond correctly for points. Scores are tracked persistently, and the top user earns a special server role.

Features include timed “Back” or “Over” challenges, decreasing point value over time, double points for baited wrong answers, and a persistent leaderboard saved to a file. Users interact via simple commands like m!chal or m!leaderboard.

Zach's Jokers (Balatro Mod)

This project was my introduction to Lua scripting and an opportunity to explore core game design principles. Using the SMODS framework for Balatro, I designed a set of custom Jokers that experiment with mechanics like probability, resource scaling, and conditional effects.

The goal wasn’t just to mod the game, but to understand how mechanics influence player behavior. I explored balance, risk-reward systems, and how randomness can create interesting choices without feeling unfair. Many Jokers are built around tradeoffs that force strategic thinking.

This was also a hands-on way to practice modular coding, iteration, and playtesting. Experimental content is included and toggleable via config for testing new ideas quickly.
